# 速导航浏览器插件

一个功能完整的网址导航浏览器插件，支持自定义分类和数据导入导出。

## 功能特点

### 导航系统
- **主类导航**：顶部或侧边栏展示主要功能分类
- **一级导航**：作为主类导航的子菜单
- **二级分类**：展示具体分类下的详细内容，支持响应式布局
- **卡片式展示**：每个网站以卡片形式展示，包含logo、名称、描述和标签

### 数据管理
- **JSON数据存储**：使用JSON格式作为数据存储标准
- **本地持久化**：浏览器重启后数据不丢失
- **数据导入**：支持通过文件选择器导入JSON格式数据
- **数据导出**：支持将当前数据以JSON文件格式下载到本地
- **默认数据**：内置常用网址数据，方便用户快速使用
- **云备份管理**：支持通过WebDAV进行数据备份和恢复
  - 手动备份数据到WebDAV服务器
  - 自动备份功能，支持每日/每周/每月备份
  - 可配置备份时间（默认凌晨0点）
  - 支持测试WebDAV连接
  - 从WebDAV恢复数据

### 图标管理
- **多API支持**：支持配置多个图标获取API，包括主API和备选API
- **默认图标类型**：支持小地球图标和文字图标两种默认图标类型
- **图标缓存**：优化的图标缓存机制，提高加载速度

### 用户体验
- **现代化UI**：符合现代浏览器插件设计规范
- **平滑动画**：导航层级之间的平滑切换动画效果
- **响应式设计**：适配不同浏览器和屏幕尺寸
- **键盘支持**：支持键盘快捷键操作
- **无障碍支持**：确保所有可交互元素可通过键盘访问

## 安装说明

### 开发模式安装

1. 下载或克隆本项目到本地
2. 打开Chrome浏览器，进入扩展管理页面（chrome://extensions/）
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择项目根目录（包含manifest.json的目录）
6. 插件安装成功，可在浏览器工具栏看到插件图标

### Firefox安装

1. 下载或克隆本项目到本地
2. 打开Firefox浏览器，进入扩展管理页面（about:addons）
3. 点击齿轮图标，选择"调试附加组件"
4. 点击"临时载入附加组件"
5. 选择项目根目录下的manifest.json文件
6. 插件安装成功，可在浏览器工具栏看到插件图标

## 使用指南

### 基本使用

1. 点击浏览器工具栏中的插件图标，打开导航面板
2. 在左侧选择主类导航（如"综合导航"、"AI类导航"等）
3. 在顶部选择一级分类
4. 在内容区选择二级分类标签
5. 点击网站卡片即可打开对应网站

### 数据管理

1. 右键点击插件图标，选择"选项"，打开设置页面
2. **导出数据**：点击"导出数据"按钮，将当前数据导出为JSON文件
3. **导入数据**：点击"选择文件"按钮，选择JSON文件，然后点击"导入"按钮
4. **重置数据**：点击"重置为默认数据"按钮，恢复内置默认数据
